# Barbell & Scale PWA v4

Talk. Train. Track.

## Included
- Installable PWA and offline support
- Spotter chat and voice commands
- Workout set logging and history
- Meal, calorie, and macro logging
- Water and body-weight tracking
- Weight trend chart and streaks
- Profile goals and local backup/restore
- Data saved locally on the device

## Upload to GitHub Pages
1. Open `8njkjt8ktr-bot/barbell-scale-app`.
2. Choose **Add file → Upload files**.
3. Unzip this package and upload all files and folders inside it.
4. Commit to `main`.
5. Open **Settings → Pages**.
6. Choose **Deploy from a branch**, then select `main` and `/ (root)`.
7. Save and wait for publication.

The app address should be:
`https://8njkjt8ktr-bot.github.io/barbell-scale-app/`

## Install on iPhone
Open the published address in Safari, tap Share, choose **Add to Home Screen**, and tap Add.
