# Barbell & Scale v1.0

## Upload to GitHub Pages
1. Unzip this package.
2. Open the `barbell-scale-app` repository.
3. Choose **Add file → Upload files**.
4. Upload everything inside the `barbell-scale-v1` folder, including the `assets` and `icons` folders.
5. Commit to `main`.
6. Open **Settings → Pages**.
7. Choose **Deploy from a branch**, then `main` and `/ (root)`.

Open the published GitHub Pages address in Safari. Do not run `index.html` directly from the iPhone Files app, because local-file browser restrictions can prevent app features from working.
